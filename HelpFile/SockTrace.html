Socket Trace v1.6

Introduction

This is a tool for capturing conversations that take place over TCP and/or UDP based sockets. It is not a sophisticated non-intrusive tool like TDIMon or Network Monitor, but is a user mode process that works by having the traffic redirected over another port or by running it as a proxy on another machine.
